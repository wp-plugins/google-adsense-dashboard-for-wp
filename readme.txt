This plugin will display your Google Adsense earnings and related reports inside your WordPress Blog, on your Administration Dashboard.

== Description ==

Google Adsense Dashboard main features:

- display detailed info and statistics about: Total Earnings, CPC, CTR, CPM, Number of Clicks and Page Views.

- select date ranges as today, yestarday, last 7 days, last 14 days and last 30 days

- custom channels performance report in a table

- ad units performance report in a table

- set a default access level. You can give access to your Google Adsense Dashboard to users like editors, authors and contributors.

- improved page loading speeds using a dedicated cache system

- has multilingual support, a POT file is available for translations. If you have a complete translation, send me the translation file or upload it to our forum and will be included in next release.

Using this plugin your data is collected in a fast and secure manner because Google Adsense Dashboard uses OAuth2 protocol and Google Adsense API.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page, which is located under Settings -> GAds Dashboard (optionally enter your API Key, Client Secret and Client ID).
4. Authorize the application using the 'Authorize Application' button
5. Go back to the plugin configuration page, which is located under Settings -> GAds Dashboard to update the final settings.

A step by step setup tutorial is available here [Google Adsense Dashboard video tutorial](http://www.deconf.com/en/projects/google-adsense-dashboard-for-wordpress/)
